# Data Types

Conforme o ECMAScript(Orgão que padroniza o JS) standard(padrão) temos 9 tipos de dados:

- Data types

  - Primitive / Primitive value
  - Structural
  - Structural Primitive

  ## Primitives

  It's not an object and is immutable

  - String
  - Numbers
  - Boolean
  - Undefined
  - Symbol
  - BigInt

  ## Structural

  - Object
    - Array
    - Map
    - Set
    - Date
    - ...
  - Function

  ## Primitive Root Structural

  - null
